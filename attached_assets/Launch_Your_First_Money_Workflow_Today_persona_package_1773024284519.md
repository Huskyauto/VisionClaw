# Non-Technical Agent Launchpad

## Promise

Go from “I don’t know where to start” to one working agent workflow in under one hour.

## 5-step launch path

1. Pick one workflow that repeats every week.
2. Define one output the agent must produce.
3. Install one skill for that workflow.
4. Run one test with real inputs.
5. Review output and lock a weekly schedule.

## Good first workflows

- Drafting weekly status update from notes.
- Turning raw research links into decision memo.
- Converting meeting notes into action checklist.

## Avoid these on first run

- Multi-agent orchestration.
- Cross-system automations with many credentials.
- “Build me a full business autopilot.”

## Success checklist

- Workflow executed once end-to-end.
- Output is usable without major rewrite.
- Owner knows exactly when to run it again.
- One next improvement is documented.
